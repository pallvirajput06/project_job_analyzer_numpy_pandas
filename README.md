# project_job_analyzer_numpy_pandas
# 📊 Data Analyst Job Market Analyzer

An interactive Job Market Analysis Dashboard built using **Python, Pandas, NumPy, Plotly, and Streamlit**. This project analyzes Data Analyst job postings to identify the most in-demand skills, salary trends, job roles, and hiring insights through an easy-to-use dashboard.

---

## 🚀 Features

- 📌 Analyze Data Analyst job postings
- 📈 Interactive charts and visualizations
- 💼 Most in-demand skills analysis
- 💰 Salary distribution insights
- 📊 Job role and hiring trend analysis
- 🔍 Clean and user-friendly Streamlit dashboard

---

## 🛠️ Tech Stack

- Python
- Pandas
- NumPy
- Plotly
- Streamlit

## ▶️ Run Locally

1. Clone the repository

```bash
git clone <your-repo-link>
```

2. Install dependencies

```bash
pip install -r requirements.txt
```

3. Run the Streamlit app

```bash
streamlit run app.py
```

## 📊 Dataset

The dataset contains Data Analyst job postings and includes information such as:

- Job Title
- Skills
- Salary
- Company
- Location

## 🎯 Project Objective

The objective of this project is to help aspiring Data Analysts understand current job market trends by analyzing real-world job postings and identifying the most valuable technical skills.
